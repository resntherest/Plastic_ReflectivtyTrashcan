
# Smart Trashcan with Plastic Detection

An Arduino-based smart trashcan that automatically detects plastic using an IR sensor and opens the lid only when plastic is detected and the trash bin is not full. The system integrates an ESP32-powered web app, ultrasonic sensors, a stepper motor, and a servo motor to control sorting and lid opening.

---

## Project Overview

This project improves waste segregation by creating a smart, automated trashcan capable of:

- Detecting plastic vs. non-plastic objects
- Checking if the trash bin is full or not
- Opening the lid and sorting items automatically
- Allowing manual control through a web interface hosted on ESP32
